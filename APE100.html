สรุปคำสั่ง HTML

ลำดับ Tag ความหมาย รูปแบบในการใส่ข้อมูล
1 "< html>" เป็นแท็กเริ่มต้นและสิ้นสุดของเอกสาร HTML < html>…< /html>
2 < head> เป็นแท็กที่ใช้กำหนดในส่วนที่เป็นชื่อเรื่อง< head> < head>…< /head>
3 < title> กำหนดข้อความบนไตเติลบาร์เป็นคำสั่งในส่วน < head>< title>ข้อความ< /title>< /head>
4 < meta> เป็นแท็กสำหรับแสดงข่าวสารของไฟล์เอกสาร HTML < meta>
5 < h1> เป็นแท็กสำหรับกำหนดหัวข้อให้เป็นตัวอักษรขนาดใหญ่สุดถึงเล็กสุดกำหนดให้ไม่เกิน 6 < h1>…< /h1>
6 < br> ขึ้นบรรทัดใหม่ < br>
7 < hr> ขีดเส้นคั่น < hr>
8 < background> ใช้รวมกับ< body>หรือ< table>เพื่อกำหนดสีของพื้นหลังเป็นรูปภาพ < body background= "ชื่อ.นามสกุลรูปภาพ">
9 < bgcolor> ใช้รวมกับ< body>หรือ< table>เพื่อกำหนดสีของพื้นหลัง < body bgcolor=ชื่อสี>
10 < font color, size, face>< /font> แสดงข้อความกำหนดสีของข้อความ กำหนดขนาดของข้อความกำหนดชนิดของข้อความ < font color=ชื่อสี size= "ขนาดกำหนดได้ตั้งแต่ 1 ถึง 7" face= "ชนิดของตัวอักษรเช่น Angsana new เป็นต้น">ข้อความ< /font>
11 < b> ข้อความหนา < b>ข้อความ< /b>
12 < i> ข้อความเอียง < i>ข้อความ< /i>
13 < u> ข้อความขีดเส้นใต้ < u>ข้อความ< /u>
14 < img src alt width height> แสดงรูปภาพ กำหนดรูปภาพ กำหนดข้อความเมื่อเอาเมาส์วางบนภาพ กำหนดความกว้างของภาพ กำหนดความยาวของภาพ < img src= "ชื่อ.นามสกุล" width= "ความกว้าง" height "ความยาว" alt= "ข้อความ">เช่น < img src = "ppp.jpg" width= "80" height "200" alt= "ภาพทะเล">
15 < tt> เป็นแท็กสำหรับกำหนดตัวอักษรเป็นตัวพิมพ์ดีด < tt>…< /tt>
16 < ul> เป็นแท็กสำหรับแสดงรายการโดยไม่ต้องเรียงลำดับ < ul>…< /ul>
17 < ol> เป็นแท็กสำหรับแสดงรายการโดยเรียงลำดับ < ol>…< /ol>
18 < li> เป็นแท็กแสดงข้อความในแต่ละบรรทัดใน แท็ก < ul>และ < ol> < li>…< /li>
19 < a> เป็นแท็กที่กำหนดการเชื่อมโยง < a>…< /a>
20 < img> เป็นแท็กที่กำหนดการเชื่อมโยงด้วยรูปภาพ < img>…< /img>
21 < a href target> กำหนดการเชื่อมโยง กำหนดจุดเชื่อมโยง กำหนดการเปิดหน้าต่างใหม่ < a href= "จุดเชื่อมโยง" target= "ลักษณะการเปิดหน้าต่าง" >ข้อความ เช่น < a href= "http://google.com" target= "_blank">facebook< /a>
22 < table> เป็นแท็กสำหรับสร้างตาราง < table>…< /table>
23 < table width, bgcolor, border, bordercolor> สร้างตารางกำหนดความกว้างของตาราง < table width = "80% bgcolor=blue border= "2" bordercolor=red>
24 < tr> เป็นแท็กสำหรับการกำหนดแถวในตาราง < tr>…< /tr>
25 < td> เป็นแท็กสำหรับกำหนดคอลัมน์ในแถว < td>…< /td>
26 < caption> เป็นแท็กสำหรับกำหนดคำอธิบายในตาราง < caption>…< /caption>
27 < th> เป็นแท็กสำหรับกำหนดข้อความหัวข้อเรื่องของตาราง < th>…< /th>
28 < form> เป็นแท็กสำหรับกำหนดสร้างform < form>…< /form>
29 < input> เป็นแท็กสำหรับกำหนด inputใน form < input>…< /input>
30 < p> กำหนดย่อหน้า < p>...< /p>
31 < !--...--> กำหนดความเห็น < !--This is a comment-->
32 < ul> แสดงรายการข้อมูลแบบแจกแจงเป็นข้อๆ โดยไม่เรียงลำดับ < ul>< /ul>
33 < style> กำหนดรูปแบบโดยใช้ style sheet < style>< /style>
34 < label> กำหนดข้อความตัวเลือกที่แสดงในลิสต์บ็อกซ์ < label for="...">... :< /label>
35 < div> แบ่งส่วนและกำหนดรูปแบบเนื้อความแบบเป็นบล็อก < div>...< /div>
36 < span> แบ่งส่วนและกำหนดรูปแบบท่อนข้อความแบบ inline < span>...< /span>
37 < mark> highlightข้อความที่ต้องการ < mark>...< /mark>
38 < sub> แสดงข้อความแบบตัวพ่วงท้าย < sub>...< /sub>
39 < sup> แสดงข้อความแบบตัวยกระดับ < sup>...< /sup>
40 < dd> แสดงส่วนคำอธิบายในรายการข้อมูลแบบให้คำนิยาม < dd>...< /dd>
